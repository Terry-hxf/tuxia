> 更新日期：2026-06-10

## 一、产品概要

**图匣（TUXIA）** 是一款免费在线图片处理工具箱，围绕「打开就能做」的理念设计。产品将图片压缩、格式转换、尺寸调整、裁剪、加水印、局部擦除与马赛克、Base64 互转、二维码生成与解码、配色提取、EXIF 元数据查看和图片转在线链接等 **12 款工具**整合为一个个清晰的工作台页面，用户无需安装任何软件，打开浏览器即可使用。

| 属性 | 值 |
|------|----|
| 产品名称 | 图匣 TUXIA |
| 标语 | 在线图片处理，打开就能做 |
| 语言 | 简体中文（zh-CN） |
| 部署地址 | `http://10.100.1.239:33103/` |
| 技术栈 | Hugo 静态站点 + 原生 JavaScript + Canvas API |
| 后端依赖 | 通用后端 API（登录/注册/反馈/内容）+ ImgBB（图片外链） |
| 许可证 | 免费使用，部分功能需登录 |

## 二、核心特点

### 2.1 本地优先处理 🏠

图匣最核心的差异化特征。除"图片转在线链接"外，其余 11 款工具的所有处理均在**浏览器本地**完成——图片不会被上传到任何服务器。这意味着：
- **隐私安全**：敏感图片（证件、合同、私密照片）不会离开你的设备
- **速度更快**：免去上传等待时间，大图处理瞬间完成
- **无需担心数据泄露**：不依赖服务端存储，不存在图片留存风险

### 2.2 工作台模式 🎛️

每个工具都是一个**完整的工作台**，上传图片后页面自动展开对应的操作区：
- 压缩工具：原图/压缩后对比面板 + 质量滑块
- 裁剪工具：Canvas 画布 + 选区拖拽 + 比例锁定
- 加水印工具：文字/图片水印 + 位置/透明度/铺排样式调节
- 取色工具：主色板展示 + 图片悬停实时取色
- Base64 工作台：左右双栏，左侧编码右侧解码

### 2.3 全流程覆盖 🔄

12 款工具覆盖了图片处理的常见链路：
```
体积优化 → 格式转换/压缩
尺寸调整 → 尺寸调整/裁剪
内容加工 → 水印/擦除/马赛克
信息提取 → 取色/EXIF/二维码解码
编码分享 → Base64/二维码生成/在线链接
```

用户可以在同一体系内连续完成多项操作，无需在多个工具网站之间跳转。

### 2.4 免安装·跨平台 🌐

- 纯 Web 应用，无需安装、注册即可使用基础功能
- 支持 PWA（Progressive Web App），可添加到桌面
- 适配移动端（390px 宽度无横向溢出）
- 兼容主流现代浏览器（依赖 Canvas API）

### 2.5 暗黑模式 🌙

完整的 Light/Dark 双主题支持，CSS 变量驱动，可在页面顶部一键切换，主题偏好存储在 `localStorage` 中持久化。

## 三、12 款工具详解

### 🔤 编码与分享类

#### 3.1 图片 Base64 工作台
- **页面**：`/tools/base64.html`
- **标签**：双栏互转
- **功能**：图片 → Base64 编码（左侧面板）、Base64 字符串 → 图片预览（右侧面板）
- **操作**：上传图片自动生成编码，支持一键复制；粘贴 Base64 字符串即时解码预览
- **场景**：前端开发中的内联图片、API 传输、Markdown 嵌入

#### 3.2 图片在线链接
- **页面**：`/tools/online.html`
- **标签**：云端分享
- **功能**：上传图片生成可分享的在线 URL（基于 ImgBB API）
- **需要**：需配置 ImgBB API Key（存储在浏览器本地）
- **场景**：临时外链、素材交付、跨设备查看
- **注意**：这是唯一需要上传图片的功能

#### 3.3 二维码生成工作台
- **页面**：`/tools/qrcode-gen.html`
- **标签**：实时定制
- **功能**：文本/链接转二维码，支持调节尺寸、容错等级、颜色、圆点样式、添加 Logo
- **导出**：PNG、SVG 格式
- **技术**：`qr-code-styling.js` 库，浏览器端渲染
- **场景**：链接分享、名片二维码、品牌定制二维码

#### 3.4 二维码解码
- **页面**：`/tools/qrcode-dec.html`
- **标签**：内容识别
- **功能**：上传二维码图片，读取其中的文本/URL
- **智能识别**：自动判断内容是纯文本还是链接，链接可直接打开
- **技术**：`jsQR.js` 库，Canvas 端解析
- **场景**：识别截图中的二维码、验证二维码内容

---

### 📐 压缩与转换类

#### 3.5 图片压缩
- **页面**：`/tools/compress.html`
- **标签**：对比压缩
- **功能**：上传图片 → 调节压缩质量（10%-100%）→ 对比原图与压缩结果 → 选择格式下载
- **支持格式**：输入 PNG/JPG/WebP（最大 2MB），输出 PNG/JPG/WebP/SVG
- **特色**：左右对比面板，清晰展示大小变化和节省比例
- **场景**：网站图片优化、邮件附件缩小、节省存储空间

#### 3.6 图片格式转换
- **页面**：`/tools/convert.html`
- **标签**：多格式导出
- **功能**：上传图片 → 预览原图 → 选择目标格式 → 下载
- **支持格式**：PNG ↔ JPG ↔ WebP ↔ BMP ↔ SVG
- **场景**：PNG 转 JPG 减小体积、JPG 转 WebP 优化 Web、转 SVG 用于矢量场景

#### 3.7 图片尺寸调整
- **页面**：`/tools/resize.html`
- **标签**：宽高缩放
- **功能**：按宽高数值或百分比缩放，支持锁定原图比例、快捷缩放档位
- **导出格式**：PNG / JPG / WebP
- **场景**：头像裁剪、电商规格图、社交媒体固定尺寸

---

### ✂️ 编辑处理类

#### 3.8 图片裁剪
- **页面**：`/tools/crop.html`
- **标签**：画布编辑
- **功能**：Canvas 画布上自由拖动选区裁剪，支持比例锁定（1:1、4:3、16:9 等）、旋转预览
- **场景**：头像裁剪、去除边框、构图调整

#### 3.9 图片加水印
- **页面**：`/tools/watermark.html`
- **标签**：本地水印
- **功能**：添加文字水印或图片水印，支持位置调节、透明度、字号、旋转角度
- **铺排模式**：单个 / 1 行 / 2 行 / 3 行 / 满屏平铺
- **特色**：所有处理在浏览器本地完成，原图不离开设备
- **场景**：版权保护、品牌标识、样图标注

#### 3.10 局部擦除 / 马赛克
- **页面**：`/tools/retouch.html`
- **标签**：修复打码
- **功能**：画笔涂抹选区 → 选择"轻量修复"或"马赛克打码" → 导出结果
- **参数**：画笔大小、马赛克粒度、模式切换
- **技术**：Canvas 像素级操作，最大支持 10MB 图片
- **场景**：去除图片瑕疵/水印/路人、隐私信息打码（证件号、人脸、手机号）

---

### 🎨 信息提取类

#### 3.11 图片配色提取
- **页面**：`/tools/palette.html`
- **标签**：实时取色
- **功能**：上传图片 → 自动提取主色板，支持鼠标悬停实时读取 HEX/RGB/HSL 色值
- **场景**：设计灵感收集、品牌色提取、UI 配色参考

#### 3.12 EXIF 元数据查看
- **页面**：`/tools/exif.html`
- **标签**：参数读取
- **功能**：上传照片 → 同步展示图片预览 + 拍摄参数（相机型号、时间、光圈、快门、ISO、焦距、尺寸、GPS 等）
- **技术**：`exif.js` 库，浏览器端解析
- **场景**：摄影参数查看、图片来源核实、GPS 定位信息读取

## 六、隐私与安全

### 数据处理原则
- **本地优先**：除"图片在线链接"外，图片不会上传到图匣服务器
- **浏览器存储**：仅保存主题偏好、登录状态、API 配置等少量状态数据
- **第三方服务**：ImgBB 接口需用户自行配置 API Key，由浏览器直接请求
- **登录要求**：所有工具功能需登录后使用；未登录时操作控件会触发登录弹窗

### 权限规则
- 后端 API 仅用于账号认证、反馈提交、备案展示和协议内容
- 图片处理不经过后端，完全在用户设备完成
- 详见 [隐私政策](/privacy.html) 和 [使用条款](/terms.html)
