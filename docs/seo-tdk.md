## 说明

- 每个工具页使用独立 `title / description / keywords`，面向具体搜索意图做长尾词覆盖。
- 页面显示标题使用 `heading` 参数，与 SEO 标题分离，避免页面 UI 过长。
- 首页输出 `WebApplication` 和 `ItemList` 结构化数据，工具页输出 `SoftwareApplication` 结构化数据。

## Hugo 构建

- 源文件: `hugo.toml`, `layouts/`, `content/`, `static/`
- 构建命令: `hugo --minify`
- 发布目录: `public/`
- 域名确定后，将 `hugo.toml` 的 `baseURL` 从 `/` 改成正式域名。
