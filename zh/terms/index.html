静态网站范围

TUXIA 是一个提供浏览器图片工具的静态前端网站。它不是账号平台、云存储服务、文件托管服务、专业编辑服务，也不是托管式图片处理服务。

本站工具用于通用图片处理任务，例如压缩、格式转换、尺寸调整、裁剪、水印、局部擦除、马赛克、Base64 转换、二维码生成和解码、取色、EXIF 元数据查看，以及通过第三方服务生成图片在线链接。

你需要对自己选择打开、处理、导出、上传到第三方或分享的图片和内容负责。你应当只使用自己拥有或有权使用的图片。请勿将这些工具用于违法、侵权、恶意、欺骗或滥用目的。

本地处理和导出

大多数工具在你的浏览器本地运行，不会把你选择的图片文件发送到 TUXIA 服务器。导出的文件由你的浏览器生成。用于生产、法律、商业或敏感流程前，请自行检查最终输出。

图片在线链接工具只有在你提供自己的 ImgBB API Key 并选择上传时，才会由你的浏览器直接把图片发送到 ImgBB。该上传行为受 ImgBB 自己的服务规则约束。TUXIA 不托管、不存储、不审核，也不删除上传到 ImgBB 的文件。
