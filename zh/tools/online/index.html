工具指南

图片在线链接生成工具指南和常见问题

使用你自己的 ImgBB API Key 将图片直接上传到 ImgBB，并生成可分享的图片链接，适合临时分享和跨设备查看。

适合使用

  • 使用你自己的 ImgBB API Key 上传图片。
  • 生成可复制、可分享的公开图片链接。
  • 保持上传路径透明：浏览器直接请求 ImgBB，不经过 TUXIA 服务器。

使用方式

  1. 在设置区域填写你的 ImgBB API Key。
  2. 选择图片并在页面中预览文件信息。
  3. 点击上传后复制返回的图片链接。
隐私说明： 这是唯一会把图片发送到第三方的工具。只有你点击上传后，浏览器才会使用你的 API Key 直接把图片发送到 ImgBB。

常见问题

需要 ImgBB API Key 吗？

需要。这个静态版不提供公共上传密钥，需要用户自行填写 ImgBB API Key。

TUXIA 会保存上传图片吗？

不会。图片从你的浏览器直接发送到 ImgBB，TUXIA 不代理、不保存。
