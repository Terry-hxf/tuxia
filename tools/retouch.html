Tool Guide

Retouch and Mosaic Tool Guide and FAQ

Use the retouch and mosaic workspace to paint over local image areas. Lightweight repair can clean small blemishes, while mosaic masking helps hide faces, license plates, QR codes, private text, and other sensitive regions before sharing.

Best for

  • Mask private areas with mosaic blocks.
  • Apply lightweight local repair to small visual distractions.
  • Export a privacy-safe copy without uploading the image.

How to use it

  1. Upload an image and choose retouch or mosaic mode.
  2. Adjust brush size and strength, then paint the target area.
  3. Preview the result and download the edited image.
Privacy note: Retouching and mosaic masking run locally in the browser. The selected image is not uploaded to a TUXIA server.

Frequently asked questions

Is the retouch feature the same as AI inpainting?

No. This static browser tool provides lightweight local repair, not cloud AI inpainting.

When should I use mosaic mode?

Use mosaic mode when you want to hide sensitive details such as faces, IDs, license plates, QR codes, or private messages.

Can I undo edits?

You can clear the selection or reset the image before exporting the final result.
