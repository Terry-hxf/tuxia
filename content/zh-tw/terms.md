## 靜態網站範圍

TUXIA 是一個提供瀏覽器圖片工具的靜態前端網站。它不是帳號平台、雲端儲存服務、檔案託管服務、專業編輯服務，也不是託管式圖片處理服務。

本站工具用於通用圖片處理任務，例如壓縮、格式轉換、尺寸調整、裁切、浮水印、局部擦除、馬賽克、Base64 轉換、QR Code 產生和解碼、取色、EXIF 中繼資料查看，以及透過第三方服務產生圖片線上連結。

你需要對自己選擇開啟、處理、匯出、上傳到第三方或分享的圖片和內容負責。你應當只使用自己擁有或有權使用的圖片。請勿將這些工具用於違法、侵權、惡意、欺騙或濫用目的。

## 本機處理和匯出

大多數工具在你的瀏覽器本機執行，不會把你選擇的圖片檔案送到 TUXIA 伺服器。匯出的檔案由你的瀏覽器產生。用於正式、法律、商業或敏感流程前，請自行檢查最終輸出。

圖片線上連結工具只有在你提供自己的 ImgBB API Key 並選擇上傳時，才會由你的瀏覽器直接把圖片送到 ImgBB。該上傳行為受 ImgBB 自己的服務規則約束。TUXIA 不託管、不儲存、不審核，也不刪除上傳到 ImgBB 的檔案。
