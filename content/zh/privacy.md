## 浏览器本地处理

TUXIA 是一个静态前端网站。图片工具主要通过浏览器能力运行，例如 JavaScript、Canvas、FileReader 和本地存储。本 GitHub 静态版本不提供 TUXIA 账号系统、会员系统、用户数据库，也不提供由 TUXIA 运营的文件存储服务。

Base64 转换、二维码生成和解码、图片压缩、格式转换、尺寸调整、裁剪、水印、局部擦除、马赛克、取色和 EXIF 元数据查看等工具，会在你的浏览器中处理所选文件。这些工具不会把你选择的图片上传到 TUXIA 服务器。

## 图片在线链接工具

图片在线链接工具的用途是生成公开图片链接，因此它和其他本地工具不同。如果你输入自己的 ImgBB API Key 并点击上传，你的浏览器会通过 ImgBB API 直接把所选图片发送给 ImgBB。TUXIA 不接收、不代理、不存储、不审核，也不控制该上传过程。使用前请阅读 ImgBB 自己的服务条款和隐私政策。

## 本地存储

TUXIA 可能会在你的浏览器中保存少量数据，例如主题偏好、语言偏好、导出格式偏好，以及你选择保存的 ImgBB API Key。这些数据保存在你的浏览器本地存储中，可以通过浏览器设置清除。

## Cookie、广告和语言页面

TUXIA 可能通过 Google AdSense 展示广告。Google 及其合作伙伴可能使用 Cookie、广告标识符、IP 地址信息、浏览器信息、设备信号和页面交互数据，用于投放、衡量和改进广告。根据你所在地区和 Google 设置，广告可能是个性化或非个性化的。

TUXIA 不运营自己的广告画像数据库，也不会把你选择的图片文件与广告数据合并。页面加载时，浏览器仍可能请求 Google 广告脚本。你可以在浏览器设置中管理 Cookie，也可以通过 Google 的广告控制选项管理广告个性化。

本站提供英语、简体中文和繁体中文静态页面。语言切换器会在这些静态页面之间跳转，并可能在你的浏览器中保存语言偏好。TUXIA 不需要通过翻译服务来处理你的图片。

## 无账号和无 TUXIA 用户追踪

本静态版本不包含登录、注册、个人资料、用户云端历史或服务端账号隔离，因为 TUXIA 在该版本中不运营后端账号服务。

## 第三方服务

本站可能加载 Google AdSense 等第三方脚本。这些服务商可能会按照其自身政策处理技术信息。如果你使用图片在线链接工具，你的浏览器会直接与 ImgBB 通信。TUXIA 不控制这些第三方服务、Cookie、保存期限或政策更新。

## 更新

当静态站点功能或说明发生变化时，本政策可能会更新。必要时，页面顶部的更新时间会同步调整。
