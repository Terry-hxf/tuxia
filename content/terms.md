## Scope of This Static Website

TUXIA is a static front-end website that offers browser-based image utilities. It is not an account platform, cloud storage service, file hosting service, professional editing service, or managed image processing service.

The tools are provided as general-purpose utilities for tasks such as compression, format conversion, resizing, cropping, watermarking, local retouching, mosaic masking, Base64 conversion, QR code generation and decoding, color extraction, EXIF viewing, and optional image URL generation through a third-party provider.

## Local Processing and Export

Most tools run locally in your browser and do not send your selected image files to a TUXIA server. Exported files are generated by your browser. Please review the final output before using it in production, legal, commercial, or sensitive workflows.

The Image URL Generator sends an image directly from your browser to ImgBB only when you provide your own ImgBB API Key and choose to upload. That upload is governed by ImgBB's own service rules. TUXIA does not host, store, moderate, or remove files uploaded to ImgBB.
