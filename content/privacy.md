## Local Browser Processing

TUXIA is a static front-end website. The image tools are designed to run in your browser by using local browser features such as JavaScript, Canvas, FileReader, and local storage. There is no TUXIA account system, membership system, user database, or TUXIA-operated file storage service in this GitHub static version.

Tools such as Base64 conversion, QR code generation and decoding, image compression, format conversion, resizing, cropping, watermarking, local retouching, mosaic masking, color extraction, and EXIF viewing process the selected files locally in your browser. These tools do not upload your selected images to a TUXIA server.

## Image URL Generator

The Image URL Generator is different because it is meant to create a public image link. If you enter your own ImgBB API Key and click the upload button, your browser sends the selected image directly to ImgBB through ImgBB's API. TUXIA does not receive, proxy, store, review, or control that upload. Please read ImgBB's own terms and privacy policy before using that feature.

## Local Storage

TUXIA may save a small amount of data in your own browser, such as theme preference, language preference, export format preference, and an ImgBB API Key if you choose to save it. This data is stored in your browser local storage and can be cleared through your browser settings.

## No Accounts or User Tracking by TUXIA

This static version does not include sign-in, registration, personal profiles, user-specific cloud history, or server-side account isolation because TUXIA does not operate a backend account service for this version.

## Third-Party Services

This site may load third-party scripts such as Google AdSense and Google Translate. Those providers may process technical information according to their own policies. If you use the Image URL Generator, your browser communicates directly with ImgBB. TUXIA does not control those third-party services.

## Updates

This policy may be updated when the static site changes. The updated date at the top of this page will be adjusted when needed.
