適合使用

  • 使用你自己的 ImgBB API Key 上傳圖片。
  • 產生可複製、可分享的公開圖片連結。
  • 保持上傳路徑透明：瀏覽器直接請求 ImgBB，不經過 TUXIA 伺服器。

使用方式

  1. 在設定區域填寫你的 ImgBB API Key。
  2. 選擇圖片並在頁面中預覽檔案資訊。
  3. 點擊上傳後複製返回的圖片連結。
隱私說明： 這是唯一會把圖片傳送到第三方的工具。只有你點擊上傳後，瀏覽器才會使用你的 API Key 直接把圖片傳送到 ImgBB。

常見問題

需要 ImgBB API Key 嗎？

需要。這個靜態版不提供公共上傳金鑰，需要使用者自行填寫 ImgBB API Key。

TUXIA 會儲存上傳圖片嗎？

不會。圖片從你的瀏覽器直接送到 ImgBB，TUXIA 不代理、不儲存。
