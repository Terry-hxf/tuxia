工具指南

圖片壓縮工具指南與常見問題

在瀏覽器中壓縮 JPG、PNG、WebP 圖片，支援原圖和壓縮後對比、品質調整和格式匯出，適合網站、郵件、文件和社群平台上傳前最佳化。

適合使用

  • 在上傳網站前減少 JPG、PNG 和 WebP 圖片大小。
  • 透過原圖和壓縮後對比檢查畫質變化。
  • 依照使用情境選擇 PNG、JPG、WebP 或 SVG 匯出。

使用方式

  1. 上傳 JPG、PNG 或 WebP 圖片。
  2. 移動品質滑桿，並查看壓縮前後對比。
  3. 選擇匯出格式並下載壓縮後的圖片。
隱私說明： 圖片壓縮使用瀏覽器 Canvas 能力在本機完成，所選圖片不會上傳到 TUXIA 伺服器。

常見問題

為什麼有些圖片壓縮後反而變大？

部分圖片已經被壓縮過，或者格式轉換產生額外開銷。可以嘗試 WebP 或進一步降低品質。

壓縮會影響畫質嗎？

有損格式會影響畫質，建議透過頁面中的前後對比預覽選擇合適品質。
