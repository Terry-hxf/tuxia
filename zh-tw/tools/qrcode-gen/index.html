工具指南

QR Code 產生工具指南與常見問題

輸入文字或連結即可產生 QR Code，支援尺寸、容錯等級、顏色、樣式、Logo 和 PNG/SVG 匯出，適合分享、印刷和營運素材。

適合使用

  • 把網址、文字或短訊息產生 QR Code。
  • 自訂 QR Code 尺寸、顏色、樣式和容錯等級。
  • 匯出 PNG 或 SVG，用於網站、印刷和營運素材。

使用方式

  1. 輸入需要編碼的文字或連結。
  2. 調整尺寸、容錯等級、顏色、樣式和可選 Logo。
  3. 預覽 QR Code 並選擇 PNG 或 SVG 下載。
隱私說明： QR Code 產生在瀏覽器本機完成，你輸入的文字不會傳送到 TUXIA 伺服器。

常見問題

QR Code 可以匯出 SVG 嗎？

可以。SVG 適合需要放大或印刷的場景。

可以加入 Logo 嗎？

可以。建議搭配較高容錯等級，並在匯出後實際掃碼測試。
