工具指南

圖片格式轉換工具指南與常見問題

支援 PNG、JPG、WebP、BMP 等常見格式互轉，上傳後可預覽並下載轉換結果，適合網站發布、文件整理和素材處理。

適合使用

  • 將 PNG 轉 JPG、JPG 轉 WebP、WebP 轉 PNG 或 BMP 轉 PNG。
  • 為網站、文件、郵件或素材交付準備合適圖片格式。
  • 匯出前先預覽來源圖片。

使用方式

  1. 上傳瀏覽器支援的圖片檔案。
  2. 選擇目標匯出格式。
  3. 預覽確認後下載轉換後的檔案。
隱私說明： 格式轉換在瀏覽器本機完成，TUXIA 不會把圖片上傳到後端服務。

常見問題

轉換格式會保留透明背景嗎？

PNG 和 WebP 支援透明背景，JPG 不支援透明背景。

SVG 匯出是向量化嗎？

不是。SVG 匯出主要是把點陣圖包裝成 SVG 檔案，不會自動變成可編輯向量路徑。
