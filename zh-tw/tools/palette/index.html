工具指南

圖片配色擷取工具指南與常見問題

從圖片中擷取主色，並在預覽圖上懸停讀取即時 HEX、RGB、HSL 色值，適合設計取色、品牌配色和前端開發。

適合使用

  • 從商品圖、截圖或設計稿中擷取主色。
  • 在圖片預覽上懸停讀取即時顏色。
  • 複製 HEX、RGB、HSL 或 CSS 變數格式。

使用方式

  1. 上傳圖片並產生配色板。
  2. 在預覽圖上移動滑鼠，查看目前位置顏色。
  3. 切換顏色格式，複製單個顏色或整組配色。
隱私說明： 取色和配色擷取都在瀏覽器本機完成，TUXIA 不會上傳你的圖片。

常見問題

支援哪些色彩格式？

支援 HEX、RGB、HSL 和 CSS 變數格式。

可以讀取滑鼠位置的顏色嗎？

可以。上傳圖片後在預覽區域懸停即可看到目前位置顏色。
