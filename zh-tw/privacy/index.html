瀏覽器本機處理

TUXIA 是一個靜態前端網站。圖片工具主要透過瀏覽器能力執行，例如 JavaScript、Canvas、FileReader 和本機儲存。本 GitHub 靜態版本不提供 TUXIA 帳號系統、會員系統、使用者資料庫，也不提供由 TUXIA 營運的檔案儲存服務。

Base64 轉換、QR Code 產生和解碼、圖片壓縮、格式轉換、尺寸調整、裁切、浮水印、局部擦除、馬賽克、取色和 EXIF 中繼資料查看等工具，會在你的瀏覽器中處理所選檔案。這些工具不會把你選擇的圖片上傳到 TUXIA 伺服器。

圖片線上連結工具

圖片線上連結工具的用途是產生公開圖片連結，因此它和其他本機工具不同。如果你輸入自己的 ImgBB API Key 並點擊上傳，你的瀏覽器會透過 ImgBB API 直接把所選圖片送到 ImgBB。TUXIA 不接收、不代理、不儲存、不審核，也不控制該上傳過程。使用前請閱讀 ImgBB 自己的服務條款和隱私政策。

本機儲存

TUXIA 可能會在你的瀏覽器中保存少量資料，例如主題偏好、語言偏好、匯出格式偏好，以及你選擇保存的 ImgBB API Key。這些資料保存在你的瀏覽器本機儲存中，可以透過瀏覽器設定清除。

Cookie、廣告和語言頁面

TUXIA 可能透過 Google AdSense 顯示廣告。Google 及其合作夥伴可能使用 Cookie、廣告識別碼、IP 位址資訊、瀏覽器資訊、裝置信號和頁面互動資料，用於投放、衡量和改善廣告。依照你所在區域和 Google 設定，廣告可能是個人化或非個人化的。

TUXIA 不營運自己的廣告輪廓資料庫，也不會把你選擇的圖片檔案與廣告資料合併。頁面載入時，瀏覽器仍可能請求 Google 廣告腳本。你可以在瀏覽器設定中管理 Cookie，也可以透過 Google 的廣告控制選項管理廣告個人化。

本站提供英文、簡體中文和繁體中文靜態頁面。語言切換器會在這些靜態頁面之間跳轉，並可能在你的瀏覽器中保存語言偏好。TUXIA 不需要透過翻譯服務來處理你的圖片。

無帳號和無 TUXIA 使用者追蹤

本靜態版本不包含登入、註冊、個人資料、使用者雲端歷史或伺服器端帳號隔離，因為 TUXIA 在此版本中不營運後端帳號服務。

第三方服務

本站可能載入 Google AdSense 等第三方腳本。這些服務商可能會按照其自身政策處理技術資訊。如果你使用圖片線上連結工具，你的瀏覽器會直接與 ImgBB 通訊。TUXIA 不控制這些第三方服務、Cookie、保存期限或政策更新。

更新

當靜態網站功能或說明發生變化時，本政策可能會更新。必要時，頁面頂部的更新時間會同步調整。
